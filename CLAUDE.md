# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Single-page portfolio for an MFA student. Built with plain HTML and CSS — no frameworks, no build tools, no preprocessors.

## Audience

Hiring managers, professors, and other students.

## Design Constraints

- Dark, minimal aesthetic
- No CSS frameworks (no Tailwind, Bootstrap, etc.)
- No JavaScript frameworks
- No CSS preprocessors

## Rules

- Ask before restructuring any files or folders
- Only build what is explicitly requested
- Do not add features, sections, or styles that weren't asked for
